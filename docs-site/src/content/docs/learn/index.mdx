## Froglet in one picture

<div class="protocol-chain">
  <div class="protocol-step"><strong>🔑 Keypair</strong><span>You generate identity locally. Nobody grants it.</span></div>
  <div class="protocol-step"><strong>📋 Descriptor</strong><span>The provider declares who it is and what it can do.</span></div>
  <div class="protocol-step"><strong>💰 Quote</strong><span>The provider prices a specific workload for a requester.</span></div>
  <div class="protocol-step"><strong>🤝 Deal</strong><span>The requester cryptographically commits to that quote.</span></div>
  <div class="protocol-step"><strong>⚙️ Execute</strong><span>The provider runs the workload under declared limits.</span></div>
  <div class="protocol-step"><strong>🧾 Receipt</strong><span>The provider signs the outcome, settlement state, and result hash.</span></div>
</div>
