## Not distributed in v0.1.0

- Hosted paid Lightning, Stripe, or x402 rails. These are v0.2.
- PyPI package.
- npm registry package.
- Homebrew formula.
- OS package-manager packages.
- Managed hosted accounts or persistent hosted identity.
