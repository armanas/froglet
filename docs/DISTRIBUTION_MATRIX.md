## Not distributed in v0.1.0

- Hosted paid Lightning, Stripe, or x402 rails. These are deferred to v0.2.
- Hosted paid execution flows on `try.froglet.dev`.
- PyPI package.
- npm registry package.
- Homebrew formula.
- OS package-manager packages.
- Managed hosted accounts or persistent hosted identity.
